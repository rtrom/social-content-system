# YouTube Shorts Metadata — Conventions

Disclosed reference for `shorts-caption`. This is SEO copy, not a social caption.

## SEO title

- Front-load the primary keyword / search term — YouTube weighs the title heavily for Shorts search and suggested.
- Readable as a headline; ~100 char hard cap but far shorter performs better.
- May include `#Shorts` but it is not required for Shorts classification.
- Keep the primary keyword consistent with the pillar's meta title (`blog-outline`) — one topic across the release.

## Description

- **First sentence** carries the keyword and states the topic — this is the highest-weighted line.
- **Body** — a short expansion of the video's message (2–4 sentences), pulled/compressed from the pillar.
- **CTA** — the brief's CTA, plus subscribe/link where relevant.
- **Links** — pillar post link, relevant internal links.

## Tags

- 5–15 keyword tags: the primary keyword, close variants, niche terms, and topic tags.
- Source trend-relevant tags from the YouTube signal specifically.
- On-topic only — irrelevant tags hurt more than they help on YouTube.
