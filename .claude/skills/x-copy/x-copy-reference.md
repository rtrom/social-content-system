# X Copy — Conventions

Disclosed reference for `x-copy`.

## Character limit

- Standard post: 280 characters. Write to it as a hard constraint — the compression is the craft.
- (Premium long-form exists but do not rely on it; write for the 280 default unless the user specifies otherwise.)

## Hook post

- The first post is the whole ballgame — it must stand alone and stop the scroll.
- Lead with the payoff or the sharpest pulled line from the pillar. No wind-up.
- Question hooks and contrarian claims travel; avoid engagement-bait phrasing if the brand voice bans it.

## Thread structure

- 3–5 posts for a compressed key point; longer only if each post genuinely earns its place.
- One beat per post. Each post readable alone, but each pulls to the next (open loops between posts).
- Numbering (1/, 2/) optional — use if it aids readability, skip if it feels mechanical.
- Final post lands the brief's CTA and can link the pillar.

## Register

- Punchier and more declarative than the other channels — X rewards a strong take stated plainly.
