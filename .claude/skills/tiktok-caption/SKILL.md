# TikTok Caption

Writes the published post text that ships with the TikTok video: the caption, the hashtags, and the on-screen-text callouts. The script is the video; this is everything the viewer reads in the post frame. Together they are one post.

Derive from the same pillar and `content-brief` anchor as the script (apply `repurposing`), and write against `brand-voice-profile.md`. The caption carries the brief's CTA in TikTok-native phrasing.

Conventions — caption length, hashtag count and mix, callout style — are in [`tiktok-caption-reference.md`](tiktok-caption-reference.md).

## Procedure

1. **Write the caption.** Short, native, hook-adjacent — it complements the video's hook, does not repeat the script. Carry the brief's CTA. Completion: caption present, within length convention, CTA included.

2. **Set hashtags.** Apply the hashtag mix from the reference (broad + niche + trend). Completion: hashtag set matches the mix strategy, count within convention.

3. **List on-screen-text callouts.** The key text overlays the editor should place on the video (distinct from the script's inline on-screen text — these are the punch callouts). Completion: callouts listed or marked not-applicable.
