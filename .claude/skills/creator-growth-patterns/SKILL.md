# Creator Growth Patterns

A distilled, running list of **principles** — what repeatedly makes niche short-form content grow. The unit is the principle, not the case study: case studies are the raw ore, principles are what gets extracted and kept. Downstream skills reason with the principles; they do not re-read the case studies.

Two branches: **consult** (a skill or the user needs the principles) and **extract** (a new creator has been researched; distil and append).

## Branch: consult

The principles below are the reference. They are a flat peer-set — apply all of them to the judgment at hand, not a convenient few. When feeding a niche evaluation or trend read, surface the principles that bear on *this* niche and say how.

### The principles

Each principle is stated as a transferable rule, with the case it came from named for traceability.

1. **Embeddedness beats reach.** A creator genuinely inside a niche outperforms a bigger generalist account entering it. Weight authentic embeddedness over raw follower count when reading signal. *(Kylie Pitts — a niche-within-a-niche, "Thursday deep cleaning vlogs" inside the broader SAHM community she already belonged to.)*

2. **One overperforming format is a signal to double down, not diversify.** When a single recurring format keeps outperforming for a creator, that is the thing to repeat, not a cue to branch out. *(Kylie Pitts — the recurring weekly-format vlog.)*

3. **Unpolished + consistent + high-frequency beats polished + sporadic** in these niches. Production value is not the lever; showing up in a real, conversational register at volume is. *(Kylie Pitts — unpolished, conversational, high posting frequency, consistent in comments.)*

4. **Monetization follows audience-content alignment; it does not precede it.** Partnerships and products should fit the audience already built and the content already made. Aligned deals landing *after* a real audience exists outperform monetizing early or off-theme. *(Kylie Pitts — went all-in only after ~69K real followers, then took brand deals aligned with her existing content.)*

### Skepticism note

Growth "methods" attached to a creator's name are often third-party affiliate products riding on that name, not documented practice. Distil principles only from what is actually documented about a creator's growth, not from courses sold about them.

## Branch: extract

A new creator has been researched. Add principles without bloating the list into case studies.

1. **Isolate what is documented.** Separate observed, sourced facts about the creator's growth from marketing narrative built around them. Discard the latter. Completion: a short list of sourced growth facts, each traceable to a real source.

2. **Distil to transferable rules.** For each fact, ask: does this generalize beyond this one creator? Keep only what transfers. Phrase each as an imperative rule a niche evaluation could apply. Completion: each new principle is stated transferably and tagged with the creator it came from.

3. **Merge, don't accrete.** Check each candidate against the existing principles. If it restates one, strengthen the existing principle's wording instead of adding a near-duplicate. Add only genuinely new principles. Completion: no two principles in the list say the same thing; the new creator is named on every principle it informed.
