# Trend Research

Turns raw platform noise into **signal** — a short, decision-ready summary of what is actually trending in the niche right now, that a content strategist can build an angle on. The job is compression: raw searches and scraped data are noise; signal is what survives filtering. The output is *not* a research dump — it is the handful of trends worth acting on this run.

## What counts as signal

Not everything trending is signal for *this* niche. A trend is signal when it is:

- **In-niche or adjacent** — relevant to the chosen niche, not platform-wide noise.
- **Rising, not peaked** — accelerating or early, not already saturated (a trend everyone has covered is noise).
- **Format-actionable** — the niche and the user can plausibly make content on it with a repeatable, low-production format (apply `creator-growth-patterns` principle 3).

A trend that fails any of these is filtered out. Reason with `creator-growth-patterns` while filtering — embeddedness and the one-overperforming-format principle both sharpen what to keep.

## Procedure

1. **Query the niche.** Pull current trend data for the niche across the target platforms — keyword/hashtag trends, rising videos, creator movement (Virlo tools where available; web search for broader context). Query for rising signal specifically, not all-time-popular. Completion: raw trend data gathered across all target platforms, not just one.

2. **Filter to signal.** Apply the three signal tests above to every candidate trend. Discard what fails. Completion: every surviving trend passes all three tests, and each discarded major trend has a one-word reason (peaked / off-niche / not-actionable) — so the filtering is auditable, not a black box.

3. **Summarize for downstream use.** Write the signal summary: for each surviving trend, a one-line description, why it is rising, and the angle hook it suggests. Keep it short — this is the input to a brief, not a report. Completion: 3–7 trends, each with description + rise reason + angle hook. If fewer than 3 survive, say so rather than padding with noise.

## Next in the chain

Once the signal summary is written, apply the `content-brief` skill to compress the chosen trend and angle into the single coherence anchor every channel writer works from.
