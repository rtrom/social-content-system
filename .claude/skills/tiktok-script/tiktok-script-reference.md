# TikTok Script — Conventions

Disclosed reference for `tiktok-script`. Load while drafting.

## Hook formulas (first 1–2 seconds)

- **Contrarian claim** — "Everyone tells you X. They're wrong."
- **Direct callout** — "If you [audience trait], stop scrolling."
- **Open loop** — pose the question the video answers; don't answer yet.
- **Result-first** — show the payoff/outcome, then "here's how."
- **In-media-res** — start mid-action, no setup.

Pick the formula that fits the pillar's angle hook. The hook is spoken *and* on-screen text simultaneously — the viewer often watches muted.

## Beat structure

1. **Hook** (0–2s) — one of the formulas above.
2. **Setup** (2–5s) — the tension or context, one line.
3. **Payoff / body** (5–25s) — the one key point, delivered in short spoken beats. One idea per beat.
4. **Button** (last 2–3s) — a close that rewards watching: a punchline, a turn, or the CTA. Avoid "follow for more" as the only close.

## Length

- Target 21–34 seconds for a single-point video; up to ~60s only if the point genuinely needs it.
- Spoken pace is fast — write tight, cut filler words.

## Sound / trend conventions

- A trending sound raises reach; note where one fits, sourced from the `trend-research` signal.
- On-screen text should track the spoken words for muted viewing, not duplicate the caption.
- Native register beats polish — conversational, first person, unscripted-feeling even when scripted.
