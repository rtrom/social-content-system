# Newsletter Copy — Conventions

Disclosed reference for `newsletter-copy`.

## Subject line

- The biggest single lever on open rate. Specific beats clever.
- ~30–50 characters reads fully on mobile; front-load the interesting word.
- Carry curiosity or a concrete benefit; match the brand voice on clickbait tolerance.
- Avoid spam-trigger patterns (ALL CAPS, excessive punctuation, "free!!!").

## Preview text

- The inbox line after the subject (~40–100 chars shown).
- Complements the subject — do not repeat it; extend the hook or add the second beat.
- If left blank, clients pull the first body line, usually badly — always write it.

## Body structure

1. **Opening hook** — a personal or contextual entry the pillar's SEO intro doesn't have. This is the expand move's payoff — the reason email feels different from the blog.
2. **Value** — the pillar's core message and one or two key points, expanded with texture (story, context, behind-the-scenes), not new claims.
3. **CTA** — the brief's ask.

## CTA placement

- Primary CTA after the value section, as a clear standalone line or button.
- Optional soft reinforcement earlier (inline link) and/or a P.S. — the P.S. is among the most-read lines in an email.
- One ask (the brief's). Reinforce it; don't add competing asks.
